# canvasContext.transform
{% version('1.9.90', 'text') %}

### 定义
使用矩阵多次叠加当前变换的方法

### 语法
```javascript
canvasContext.transform(scaleX, skewX, skewY, scaleY, translateX, translateY)
```

### 参数
| 属性值      | 类型   | 说明          |
|------------ |--------| ------------- |
| scaleX      | Number | 水平缩放 |
| skewX       | Number | 水平倾斜 |
| skewY       | Number | 垂直倾斜 |
| scaleY      | Number | 垂直缩放 |
| translateX  | Number | 水平移动 |
| translateY  | Number | 垂直移动 |
